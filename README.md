# InfantJS

A programming language for *literal* babies

## Story

We were inspired by our professor when he decided to make a programming language called "Baby.js" in our compilers class.
We decided to take that literally. So we created a langauge based on babies most commonly used words for all babies to understand.

## How do I run this language?

You must run the translator which will automatically translate to javascript and run it.

An example of how to run/use the translator: `npm run translate examples/guess-number-game.infant`
